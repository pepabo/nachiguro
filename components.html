Avatar

ある箇所が特定のユーザーが所有するものであることを表す場合に使用します。大きさはそのセクションでの重要度に合わせて「Small・Medium・Large」の3種類を使い分けます。

ユーザー名が長すぎる場合は一覧画面など次のアクションでユーザー名が取得できる場合に限り、三点リーダーで省略することができます。

さらにユーザー名を伝えることの優先順位が低いと判断できる場合は、ユーザー名自体を省略することも可能です。

Button

.ncgr-button を指定した要素はボタンになります。

<button class="ncgr-button">タップ可能</button>

.ncgr-button--active

.ncgr-button を指定した要素に .ncgr-button--active をさらに指定すると、タップ中のボタンになります。

<button class="ncgr-button ncgr-button--active">タップ中</button>

.ncgr-button--fluid

.ncgr-button を指定した要素に .ncgr-button--fluid をさらに指定すると、ボタンの幅が内容によって可変します。

<button class="ncgr-button ncgr-button--fluid">タップ可能</button>

.ncgr-button--small

.ncgr-button を指定した要素に.ncgr-button--small を指定すると小さいサイズのボタンになります。

<button class="ncgr-button ncgr-button--small">タップ可能</button>

.ncgr-button--light

.ncgr-button を指定した要素に .ncgr-button--light を指定すると、色が薄いボタンになります。

<button class="ncgr-button ncgr-button--light">タップ可能</button>

.ncgr-button--disabled

.ncgr-button を指定した要素に disabled または .ncgr-button--disabled を指定すると、タップできないボタンになります。

甲賀流（こうかりゅう）とは、近江国甲賀の地に伝わっていた忍術流派の総称。
山を一つ隔てた場所に存在する伊賀流と並び、最も有名な忍術の一派として知られる。
甲賀流伊賀流ともに伝系に関しては仮託の説が多く信用できるものはあまりない。
しかし、民家で史料が発見されるなど研究も進み、少なくとも江戸時代には確実に存在したとされている。
なお、「甲賀流」という名称の単一の流派は存在せず、あくまで甲賀に伝わる複数の流派があわさって甲賀流と呼ばれているにすぎない。
伊賀流（いがりゅう）は、伊賀国の地に伝わっていた忍術流派の総称。
甲賀流と並んで忍術の中で最も有名な流派の一つである。
根拠地は、現在の三重県伊賀市と名張市にあった。
普段は農業や行商をして各地の情報を探る一方、指令が下ると戦場やその後方へ出向き、工作活動に励んだ。

Hint

このコンポーネントを使うことで忍者スリスリくんというキャラクターを通してユーザーにコンテンツに対する一言コメントや、SUZURIを使う上でのヒントを教えます。

ヒントは内容がないものから実用的なものまで様々に用意しています。 「https://suzuri.jp/hint」にアクセスすることで確認できます。 また、内容を変更する場合はhttps://git.pepabo.com/suzuri/suzuriのconfig/locales/ja.hint.yml、config/locales/en.hint.yml、を修正します。

Icon Button

Icon Buttonはある商品やユーザー情報に対してアクションをするボタンに使用します。

.ncgr-icon-button--black、.ncgr-icon-button--whiteのModifierをつけるとボタンに色がつきます。

「Whiteボタン」はログインしているユーザー以外である他者からのアクション（ズッキュン・オモイデ追加）がメインである場合に使用します。

<button class="ncgr-icon-button ncgr-icon-button--white">
  <i class="fa fa-folder"></i>
</button>

「Blackボタン」はログインしているユーザー自身（管理者）のアクション（編集・削除）に使います。

<button class="ncgr-icon-button ncgr-icon-button--black">
  <i class="fas fa-pencil-alt"></i>
</button>

ズッキュンボタンはエモーショナルな気持ちを表現するために暖色系の色と♥が動くアニメーションを使用します。

<button class="ncgr-icon-button ncgr-icon-button--favorite">
  <i class="fa fa-heart"></i>
  <span class="ncgr-pad-l-4">246</span>
</button>

プロフィール画像やカバー画像などの画像をアップロードする箇所には「Roundedボタン」を使用します。

<button class="ncgr-icon-button ncgr-icon-button--black ncgr-icon-button--circle">
  <i class="fas fa-pencil-alt"></i>
</button>

優先順位の低いアクションなどには「Transparentボタン」を使用します。

<button class="ncgr-icon-button ncgr-icon-button--transparent">
  <i class="fa fa-pencil-alt"></i>
</button>

Textfield

テキストフィールドのラベルはモバイルにおける省スペース化のため、テキストフィールドの枠上部に重なる形で配置します。 Material Designではラベルがプレースホルダーの位置から上部にアニメーションしながら移動しますが、ラベルとプレースホルダーの意味付けが曖昧になるため、アニメーションさせません。 SUZURIではあらゆるプラットフォームで同様の体験を提供するため、Web、iOS、Androidすべてのテキストフィールドは基本的にこのコンポーネントを使用します。

textfield同士の間隔は基本的に.ncgr-mar-b-16のutilityで調整し、イレギュラーな場合のみ他のサイズのmarginのutilityで調整します。

また、何を入力するかわかりやすいよう、記入例を例）〇〇の形でplaceholderに記入します。

また、textareaの高さは指定せず、rowsで指定します。基本的にはrows="6"で指定します。

<div class="ncgr-textfield ncgr-mar-b-16">
  <label class="ncgr-textfield__label" for="name">名前</label>
  <input class="ncgr-textfield__input" type="text" name="name" placeholder="例）スリスリ君">
</div>
<div class="ncgr-textfield ncgr-mar-b-16">
  <label class="ncgr-textfield__label" for="name">自己紹介</label>
  <textarea class="ncgr-textfield__input" name="name" rows="6" placeholder="例）忍ぶれど 色に出でにけり SUZURIです。"></textarea>
</div>

単体で使う小さなテキストフィールドを表示する場合は以下のようにマークアップします。

<div class="ncgr-textfield">
  <input class="ncgr-textfield__input ncgr-textfield__input--small" type="text" name="name" placeholder="オモイデの名前">
</div>

error・successの場合は以下のようにマークアップします。

error

<div class="ncgr-textfield ncgr-mar-b-16">
  <label class="ncgr-textfield__label" for="name">名前</label>
  <input class="ncgr-textfield__input invalid" type="text" name="name" placeholder="例）スリスリ君">
</div>

success

<div class="ncgr-textfield ncgr-mar-b-16">
  <label class="ncgr-textfield__label" for="name">名前</label>
  <input class="ncgr-textfield__input valid" type="text" name="name" placeholder="例）スリスリ君">
</div>
